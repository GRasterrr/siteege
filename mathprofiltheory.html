Практика
  • Произвольный (любой) треугольник
    Сумма углов = 180 градусов
    Площадь через высоту: \(S=\frac{1}{2}ah\)
    Площадь ч-з стороны и синус между ними: \(S=\frac{1}{2}ab\sin{a}\)
    Площадь ч-з формулу Герона (p - полупериметр): \(S=\sqrt{p(p-a)(p-b)(p-c)}\)
    Площадь ч-з радиус описанной окружности: \(S=\frac{abc}{4R}\)
    Площадь ч-з радиус ВО: \(S=rp\)
    Теорема синусов: \(\frac{a}{\sin{a}}=\frac{b}{\sin{b}}=\frac{c}{\sin{c}}=2R\)
  • Равнобедренный треугольник
    Углы при основании равны
    Биссектриса = Медиана = Высота
    Боковые стороны равны
  • Равносторонний треугольник
    Углы по 60 градусов
    Все стороны равны
  • Прямоугольный треугольник
    Один из углов = 90 градусов (прямой)
    Напротив прямого угла - наибольшая сторона (гипотенуза), остальные - катеты
    Центр описанной окружности лежит на середине гипотенузы
    Теорема Пифагора: \(c^{2}=a^{2}+b^{2}\)
    Площадь: \(S=\frac{1}{2}ab\)
    Тригонометрические соотношения: \(\sin{A}=\frac{a}{c}\), \(\cos{A}=\frac{b}{c}\), \(tg{A}=\frac{a}{b}\)
    Радиус описанной окружности: \(R=\frac{c}{2}\)
    Радиус вписанной окружности: \(r=\frac{a+b-c}{2}\)
  • Тригонометрия
    Главное тригонометрическое тождество: \(\sin^{2}{a}+\cos^{2}{a}=1\)
  • Синус
    Синус - это отношение противолежащего катета к прилежащему
    Теорема синусов: \(\frac{a}{\sin{a}}=\frac{b}{\sin{b}}=\frac{c}{\sin{c}}=2R\)
    Синус из косинуса: \(\sin{a}=\sqrt{1-\cos{a}}\)
